Policy Statement

Through the usage of the Community Events plugin, the following information may be collected and stored within your website’s database:

  • Venue information: name, address, city, country, state, postal code, phone, website
  • Organizer information: name, phone, website, email
  • Event information: website, cost, description, date, time, image
  • Please note: to create new events through the Community Events submission form, a user must hold a website account on this domain. This information is stored within WordPress core, within this website's database. It is also possible to create events anonymously, if the site owner has this option enabled.
